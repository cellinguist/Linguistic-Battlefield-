# Linguistic Battlefield
Проект посвящён анализу позиции и структуры аннотаций к лингвистическим статьям с использованием трансформеров, рекуррентных нейросетей и больших языковых моделей.

Корпус текстов представляет собой 210 аннотаций к лингвистическим статьям (1398 предложений). В корпус входят три темы: статьи о когнитивном преимуществе билингвов (bilingualism.txt, 510 предложений), статьи о гипотезе лингвистической относительности (relativity.txt, 467 предложений), статьи об универсальной грамматике (universal grammar.txt, 390 предложений). 

Каждое предложение размечено по пяти параметрам:
1. Релевантность предложения: 1 - тема однозначно определяется без контекста (ключевые слова темы); 0 - тема может быть определена без контекста (релевантные слова); -1 - тема непонятна без контекста.
2. Тип предложения: 1 - введение, обзор литературы; 2 - метод; 3 - результаты, обсуждение.
3. Позиция: 0 - против; 1 - нейтральная/противоречивая; 2 - за; -1 - нерелевантно.
4. Наличие аргумента: 1 - есть аргумент; -1 - нет аргумента.
5. Номер аннотации, из которой взято предложение.

Полная разметка представлена в файлах data_bilingualism.csv, data_linguistic_relativity.csv, data_universal_grammar.csv.

В данном проекте мы сфокусировались на параметрах "Тип предложения" и "Позиция".

Для предсказания типа предложения и позиции были применены три группы методов: трансформерные сети (RoBERTa large, SciBERT), BiLSTM-сети (без предобучения; без предобучения, но с предобученными эмбеддингами из BERT; предобученные на корпусе Dr.Inventor [Lauscher et al., 2018]), LLM (Qwen 2.5-7B-Instruct, zero-shot и few-shot). Код представлен в файле linguistic battlefield_code_final.ipynb.

Наилучшие результаты показало сочетание SciBERT с логистической регрессией с контекстом для типов предложений, особенно для выявления типа. Наиболее нестабильны результаты BiLSTM сети - хотя она лучше справлялась с предсказанием позиции. Метод фью-шота с LLM значительно улучшил только вычленение позиции. Классификация статей по когнитивному преимуществу билингвов оказалась наиболее легко выполняемой, а задача выявления типа в принципе решалась более успешно, чем выявление позиции.
