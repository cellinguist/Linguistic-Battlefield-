# Linguistic-Battlefield-
Проект посвящён анализу позиции и структуры аннотаций к лингвистическим статьям с использованием трансформеров, рекуррентных нейросетей и больших языковых моделей.

Корпус текстов представляет собой 210 аннотаций к лингвистическим статьям (1398 предложений). В корпус входят три темы: статьи о когнитивном преимуществе билингвов (bilingualism.txt, 510 предложений), статьи о гипотезе лингвистической относительности (relativity.txt, 467 предложений), статьи об универсальноц грамматике (universal grammar.txt, 390 предложений). В аннотации размечались тип каждого предложения (1 - введение, 2 - методы, 3 - результаты) и позиция автора в каждом предложении (2 - за, 0 - против, 1 - другое, -1 - позиция не выражена). Полная разметка представлена в файле annotated_data.xlsx.
Для предсказания типа предложения и позиции были применены три группы методов: трансформерные сети (RoBERTa large, SciBERT), BiLSTM - сети (без предобучения и предобученные на корпусе Dr.Inventor [Lauscher et al., 2018]), LLM (Qwen 2.5-1.5 Instruct, zero-shot и few-shot). Код представлен в файле linguistic battlefield_code.ipynb.
Наилучшие результаты показало сочетание SciBERT с логистической регрессией с контекстом для типов предложений, хуже всего справились BiLSTM сети. Классификация статей по билингвизму оказалась наиболее простой.
